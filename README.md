# Slate Editor

A Custom Document Editor with React and Slate.js
==================================


HOW TO RUN
========
1. Clone the project from git.
2. CD to 'slate-editor' directory.
3. Run `npm install` or `yarn`.
4. Run `npm start`.

Application will start at host: 'http://localhost:8081'
